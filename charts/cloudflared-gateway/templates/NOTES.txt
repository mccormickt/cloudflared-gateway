{{ .Chart.Name }} {{ .Chart.AppVersion }} has been installed into namespace "{{ .Release.Namespace }}".

1. Verify the controller pod is running:

    kubectl -n {{ .Release.Namespace }} get pods -l app.kubernetes.io/instance={{ .Release.Name }}
    kubectl -n {{ .Release.Namespace }} logs -l app.kubernetes.io/instance={{ .Release.Name }} -f

2. Create a GatewayClass that claims this controller:

    apiVersion: gateway.networking.k8s.io/v1
    kind: GatewayClass
    metadata:
      name: cloudflared
    spec:
      controllerName: {{ .Values.controllerName }}

3. Create a Gateway and routes attached to it. Once reconciled, the controller
   provisions a Cloudflare Tunnel and deploys `cloudflared` pods in the Gateway's
   namespace. Example Gateway + HTTPRoute manifests live in the repo under
   `examples/`.

4. Find the provisioned tunnel in the Cloudflare dashboard:

    https://one.dash.cloudflare.com/ -> Networks -> Tunnels
